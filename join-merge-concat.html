concat() method is called on pandas object and not dataframe. It accepts a list of dataframes or series which are to be concatenated and concatenates as per sequence of the passed list. Another essential arguement of concat() method is "ignore_index" when set true pandas reproduces indices considering the increase in rows after concatenating.

Here the use of this method is quite intuitive,the dataframes are merged to one dataframe. Let me explain the use of different parameters passed, "on" takes label of a column on basis of which merge is to be done. One thing is to be noted for using this parameter both dataframes should have the column names excatly same, if not there is other options that i have discussed below. You must have noticed duplication of some "Customer ID", this is because same customer came multiple number of times which is evident when i did that filteration of data.

Here we are applying "left join", that is trying to get data from week_1 excluding week_2, as week_2 is right df here.
